# Open Source Monetization
Law IV: Default to Open doesn’t mean broke.

**3 paths that work:**
1. **Support/Consulting**: Charge for helping people use your open code
2. **Hosted Service**: Code is free, running it at scale isn’t 
3. **Reputation → Opportunity**: Open code = proof. Proof = clients, jobs, deals

Rep > Equity. Give away the code. Sell the execution.
